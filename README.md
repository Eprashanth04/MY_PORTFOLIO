# Prashanth Portfolio

A personal portfolio built with Vite, React, TypeScript, Tailwind CSS, and shadcn-ui.

## Development

Prerequisites:

- Node.js and npm

Install dependencies and start the dev server:

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
npm run preview
```

## Tech Stack

- Vite
- React (TypeScript)
- Tailwind CSS
- shadcn-ui
